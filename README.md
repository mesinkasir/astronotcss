Get started

insert this code on your template 

`<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/astronot@1.0.1-sv/astronot.min.css"/>`

before `</head>`

or run

`npm i astronot`

run lastest version

`npm i astronot@1.0.1`
