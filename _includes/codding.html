Astronot CSS framework terbaru persembahan dari axcora technology team developer untuk dunia.

Memulai menggunakan nya sekarang dengan CDN js insert code berikut ini sebelum tag head pada template code website blog mu.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/astronot/css/astronot.min.css"/>

Atau install dengan menggunakan NPM pada terminal jalankan perintah berikut ini

npm install astronot
